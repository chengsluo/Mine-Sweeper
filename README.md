# Mine-Sweeper
这是一个在Windows运行下的扫雷程序，由Win32实现
## 游戏说明
游戏预设分高中低三个等级的游戏，另外也可以由玩家预设地图长宽和地雷数量。

游戏分为计时模式和不计时模式，在自定义模式中可以选择
